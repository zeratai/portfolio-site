Kidoodle.TV Android Application

Android Application through Android Studio

I also worked on creating smoother animations and image loading for transitions amongst different devices such as: mobile, tablet, and smartTV.

You can preview the Kidoodle.TV app here.

Destiny Armory Database

Database Java Project expanded with GUI and Android APK

This was my CS166 Database Project written in Java and using Postgres as the database. I used postressql library along with Android Studio to create this simple Destiny database application.

The application's user interface and design was created in Photoshop and ported using XML and multi-threaded programming to render images and query the database.

You can check out the source code at my GitHub.
